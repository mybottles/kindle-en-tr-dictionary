# Katkı Rehberi

Bu projeye katkıda bulunmak için:

## Yanlış/eksik çeviri bildirimi

Issue aç, `translation` etiketi kullan. Şablonu doldur.

## Kod katkısı

1. Fork et
2. Yeni branch aç: `git checkout -b feature/aciklama`
3. Değişikliği yap, test et: `python build.py` çalışıyor mu?
4. Commit at, push et
5. Pull request aç

## Veri kaynağı önerisi

Yeni bir açık lisanslı İngilizce-Türkçe veri kaynağı biliyorsan Issue aç. Lisansı (CC, GPL, public domain) ve indirilebilir URL'i ekle.

## Test

Kindle Paperwhite 12 üzerinde test edebiliyorsan ideal. Daha eski Kindle'lar için test edebilirsen `test reports` etiketli Issue'lara raporla.
